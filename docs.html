Requirements

Sebelum menginstall script dari ExterStore, pastikan server kamu memenuhi persyaratan berikut:

Server Requirements

Requirement Minimum Recommended
FiveM Server Latest Artifact Latest Artifact
Framework ESX 1.2+ / QBCore ESX Legacy / QBCore Latest
Database MySQL 5.7+ MySQL 8.0+ / MariaDB 10.5+
oxmysql v2.0+ Latest Version

Dependencies

server.cfg 
# Required Dependencies
ensure oxmysql
ensure es_extended  # atau qb-core
ensure ox_lib       # recommended

# ExterStore Scripts
ensure ExterStore-hud
ensure ExterStore-inventory
ensure ExterStore-admin

Installation

Ikuti langkah-langkah berikut untuk menginstall script ExterStore di server kamu:

1

Download Script

Download script dari Tebex setelah melakukan pembelian. Kamu akan mendapatkan file .zip yang berisi semua file yang diperlukan.

2

Extract Files

Extract file zip ke folder resources di server FiveM kamu.

Terminal 
# Extract ke folder resources
cd /path/to/server/resources/[ExterStore]
unzip ExterStore-script.zip
3

Import Database

Import file SQL yang disertakan ke database server kamu.

SQL 
-- Import via phpMyAdmin atau command line
mysql -u root -p your_database < sql/install.sql
4

Configure & Start

Edit file config.lua sesuai kebutuhan server kamu, lalu tambahkan ke server.cfg.

server.cfg 
# Add to your server.cfg
ensure ExterStore-scriptname

Basic Configuration

Setiap script ExterStore memiliki file config.lua yang bisa kamu customize sesuai kebutuhan.

Config Structure

config.lua 
Config = {}

-- Framework Detection (auto-detect)
Config.Framework = 'auto' -- 'esx', 'qbcore', or 'auto'

-- General Settings
Config.Debug = false
Config.Locale = 'id' -- 'en', 'id', 'de', etc.

-- UI Settings
Config.Theme = 'dark' -- 'dark' or 'light'
Config.AccentColor = '#2ECC71'
Config.Font = 'Inter'

-- Performance
Config.UpdateInterval = 500 -- ms
Config.MaxDistance = 50.0 -- render distance

-- Notifications
Config.NotifySystem = 'ox_lib' -- 'esx', 'qb', 'ox_lib', 'custom'
Penting!

Jangan ubah file selain config.lua kecuali kamu tahu apa yang kamu lakukan. Perubahan pada file lain bisa menyebabkan script tidak berfungsi.

Database Setup

Beberapa script memerlukan tabel database. File SQL sudah disertakan di setiap script.

Auto Migration

Script ExterStore mendukung auto-migration. Saat pertama kali dijalankan, tabel akan dibuat secara otomatis jika belum ada.

Example SQL 
CREATE TABLE IF NOT EXISTS `ExterStore_data` (
    `id` INT AUTO_INCREMENT PRIMARY KEY,
    `identifier` VARCHAR(60) NOT NULL,
    `data` LONGTEXT DEFAULT NULL,
    `created_at` TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    `updated_at` TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    INDEX `idx_identifier` (`identifier`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;

Permissions

Konfigurasi permission untuk mengontrol akses ke fitur-fitur tertentu.

config.lua - Permissions 
Config.Permissions = {
    -- Admin Panel Access
    AdminPanel = {
        enabled = true,
        groups = {'admin', 'superadmin', 'god'},
        acePermission = 'ExterStore.admin'
    },

    -- Moderator Access
    ModPanel = {
        enabled = true,
        groups = {'moderator', 'admin', 'superadmin'},
        acePermission = 'ExterStore.moderator'
    }
}

-- ACE Permission Setup (add to server.cfg)
-- add_ace group.admin ExterStore.admin allow
-- add_ace group.moderator ExterStore.moderator allow

Custom HUD

HUD modern dan customizable dengan berbagai widget yang bisa diatur sesuai keinginan.

Configuration

config.lua - HUD Settings 
Config.HUD = {
    -- Widget Visibility
    ShowHealth = true,
    ShowArmor = true,
    ShowHunger = true,
    ShowThirst = true,
    ShowStamina = true,
    ShowOxygen = true,

    -- Minimap
    ShowMinimap = true,
    MinimapShape = 'circle', -- 'circle' or 'square'

    -- Speedometer
    ShowSpeedometer = true,
    SpeedUnit = 'kmh', -- 'kmh' or 'mph'

    -- Position

    ArmorColor = '#3498DB',
    HungerColor = '#E67E22',
    ThirstColor = '#9B59B6',
}

Inventory System

Sistem inventory modern dengan drag & drop, weight system, dan crafting.

Configuration

config.lua - Inventory Settings 
Config.Inventory = {
    -- General
    MaxWeight = 120.0, -- kg
    MaxSlots = 40,

    -- Hotbar
    HotbarSlots = 5,
    HotbarKey = 'TAB',

    -- Drop System
    EnableDrop = true,
    DropDespawnTime = 300, -- seconds

    -- Crafting
    EnableCrafting = true,
    CraftingLocations = {
        {coords = vector3(100.0, 200.0, 30.0), label = 'Workbench'},
    },

    -- Shops
    EnableShops = true,
}

Troubleshooting

Solusi untuk masalah umum yang mungkin kamu temui:

Script tidak mau start?

Pastikan semua dependencies sudah terinstall dan di-ensure sebelum script ExterStore di server.cfg.

Common Issues

Error: "oxmysql not found"

Solusi: Install oxmysql dan pastikan di-ensure sebelum script ExterStore.

Fix 
# Pastikan urutan di server.cfg:
ensure oxmysql
ensure es_extended
ensure ExterStore-scriptname

UI tidak muncul

Solusi: Clear cache browser NUI dan restart resource.

Fix 
# In-game console (F8):
restart ExterStore-scriptname

# Atau clear NUI cache:
# Delete folder: cache/browser/ di folder FiveM
